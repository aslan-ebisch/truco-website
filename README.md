# Pintura Co.

**Live at [pinturaco.com](https://pinturaco.com)**

A student-run painting company website for Northwest Arkansas and Springfield, MO.

## About

Pintura is a student-run painting company where college students gain real leadership experience while delivering professional results. Founded by a college student, the company provides crew managers with hands-on experience in team leadership, project management, and client communication.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Hosting:** Vercel

## Services

- Interior Painting
- Exterior Painting
- Wood Staining
- Pressure Washing

## Running Locally

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Deployment

Deployed on Vercel with automatic deploys from the `main` branch. Push to GitHub and the site updates automatically.
